# Commands

This section describes all commands supported by the current DCR version.

## Overview

- `dcr new <name>` - create a new project in a subdirectory.
- `dcr init` - initialize the current empty directory.
- `dcr build [--debug|--release]` - build `src/main.c`.
- `dcr run [--debug|--release]` - build and run `target/<profile>/main`.
- `dcr clean [--debug|--release]` - remove `target` entirely or by profile.
- `dcr --update` - update binary to the latest release.
- `dcr --help` - show help.
- `dcr --version` - show version and target.
