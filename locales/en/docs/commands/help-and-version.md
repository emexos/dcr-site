# dcr --help / --version

Top-level utility flags.

## dcr --help

```bash
dcr --help
```

Shows available commands, flags, and examples.

## dcr --version

```bash
dcr --version
```

Prints version and target in this format:

```text
dcr <version> (<target>)
```

## Running without a command

If you run `dcr` without arguments, it prints help equivalent to `dcr --help`.
