# Getting started

This section follows the DCR workflow from installation to first run.

- [Installation](./installation/)
- [First steps](./first-steps/)

## What you will get

- Installation on Linux, macOS, or Windows.
- Verification that `dcr` is available in your terminal.
- Creation of a minimal C project and running it with `dcr run`.

## Recommended order

1. Complete [Installation](./installation/).
2. Complete [First steps](./first-steps/).
3. Open the [Commands](../commands/) section for detailed CLI behavior.
