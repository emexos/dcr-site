# Структура проекта

## Минимальная структура

```text
project/
- dcr.toml
- src/
- - main.c
```

### Назначение директорий и файлов

- `dcr.toml` - файл конфигурации проекта.
- `src/main.c` - входной исходник, который сейчас компилирует DCR.

## Расширеная структура

```text
project/
- dcr.toml
- dct.lock
- src/
- - main.c
- target/
- - debug/
- - - main
- - release/
- - - main
```

### Назначение директорий и файлов

- `dcr.toml` - файл конфигурации проекта.
- `dct.lock` - файл блокировки зависимостей.
- `src/main.c` - входной исходник, который сейчас компилирует DCR.
- `target/debug/main` - скомпилированный проект в режиме отладки.
- `target/release/main` - скомпилированный проект в режиме релиза.
