# Платформы и установка

## Поддерживаемые target в релизах

- `x86_64-unknown-linux-gnu`
- `x86_64-apple-darwin`
- `aarch64-apple-darwin`
- `x86_64-pc-windows-msvc`

## Скрипты установки

- `install.sh` для Linux/macOS.
- `install.ps1` для Windows.

Оба скрипта предлагают:

1. Скачать готовый бинарник из релиза.
2. Или собрать из исходников.

## Пути установки

- Linux/macOS: бинарник ставится в пользовательскую директорию и линкуется в `~/.local/bin`.
- Windows: бинарник копируется в пользовательский каталог и дублируется в `~/.local/bin`.

## Обновление после установки

Для установленного DCR можно использовать:

```bash
dcr --update
```

Команда обновляет бинарник в месте, где запущен текущий `dcr`.
Или обновитьчь через установочный скрипт.
