# Конфигурация dcr.toml

Основной файл проекта - `dcr.toml`.

## Минимальный пример

```toml
[package]
name = "hello"
version = "0.1.0"
language = "c"
compiler = "clang"

[dependencies]
```

## Что записывает DCR

Команды `dcr new` и `dcr init` создают `dcr.toml` именно в таком базовом виде.

## Текущее поведение

- Для команд `build`, `run`, `clean` файл `dcr.toml` обязателен.
- На текущем этапе значения из `dcr.toml` не участвуют в выборе файла для сборки.
- Реальная компиляция ориентирована на `src/main.c` и константы в коде DCR.

## Практика

Считайте `dcr.toml` маркером проекта и точкой для будущего расширения конфигурации.
