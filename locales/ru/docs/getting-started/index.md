# Начало работы

Этот раздел повторяет рабочий путь DCR от установки до первого запуска.

- [Установка](./installation/)
- [Первые шаги](./first-steps/)

## Что вы получите

- Установку на Linux, macOS или Windows.
- Проверку, что `dcr` доступен в терминале.
- Создание минимального C-проекта и запуск через `dcr run`.

## Рекомендуемый порядок

1. Пройти [Установку](./installation/).
2. Пройти [Первые шаги](./first-steps/).
3. Открыть раздел [Команды](../commands/) для подробного поведения CLI.
