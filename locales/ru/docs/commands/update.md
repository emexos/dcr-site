# dcr --update

Обновляет установленный бинарник DCR до последнего релиза из GitHub.

## Использование

```bash
dcr --update
```

## Как это работает

1. Запрашивается `https://api.github.com/repos/dexoron/dcr/releases/latest`.
2. Берется `tag_name` последнего релиза.
3. Подбирается ассет по текущему target (`DCR_TARGET`).
4. Новый бинарник скачивается во временный файл.
5. Текущий исполняемый файл заменяется через `self-replace`.

## Когда обновление не требуется

Если версия уже актуальная, команда сообщает об этом и завершается без изменений.

## Ограничения

- Команда не принимает аргументы.
- Требуется доступ к GitHub API и ассетам релиза.
- Для Windows ожидается ассет `dcr-x86_64-pc-windows-msvc.exe`.
