# dcr run

Собирает проект и запускает бинарник.

## Использование

```bash
dcr run
dcr run --debug
dcr run --release
```

## Что делает команда

1. Проверяет наличие `dcr.toml`.
2. Запускает ту же сборку, что и `dcr build`.
3. Выполняет `./target/<profile>/main`.

## Поведение при ошибках сборки

- Сначала команда пытается собрать проект.
- Если сборка не удалась, команда пробует запустить последний бинарник выбранного профиля.
- Если подходящего бинарника нет, выводится сообщение об ошибке.

## Ограничения

- Как и в `build`, поддерживается только один профильный флаг первым аргументом.
