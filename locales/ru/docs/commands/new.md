# dcr new

Создает новый проект в отдельной директории.

## Использование

```bash
dcr new <name>
```

## Что делает команда

1. Создает директорию `<name>`.
2. Создает файл `dcr.toml` с секциями `[package]` и `[dependencies]`.
3. Создает `src/main.c` с шаблоном `Hello World`.

## Поведение и ошибки

- Если имя не передано, команда завершится с ошибкой.
- Если передано больше одного аргумента, команда завершится с ошибкой.
- Если директория `<name>` уже существует, команда подскажет использовать `dcr init`.

## Пример

```bash
dcr new app
cd app
dcr run
```
