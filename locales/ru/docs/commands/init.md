# dcr init

Инициализирует DCR-проект в текущей директории.

## Использование

```bash
dcr init
```

## Что делает команда

1. Проверяет, что текущая директория пустая.
2. Берет имя директории как `package.name`.
3. Создает `dcr.toml`.
4. Создает `src/main.c`.

## Ограничения

- Команда не принимает аргументы.
- Если директория не пустая, команда завершится с ошибкой.

## Пример

```bash
mkdir hello
cd hello
dcr init
dcr run
```
