# dcr clean

Удаляет артефакты сборки.

## Использование

```bash
dcr clean
dcr clean --debug
dcr clean --release
```

## Что удаляется

- Без аргументов: вся директория `target`.
- С профилем: только `target/debug` или `target/release`.

## Проверки

- Нужен файл `dcr.toml` в текущей директории.
- Если `target` или указанный профиль не найден, выводится предупреждение.

## Особенность аргументов

Команда ожидает максимум один аргумент-профиль.
