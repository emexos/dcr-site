# dcr --help / --version

Служебные флаги верхнего уровня.

## dcr --help

```bash
dcr --help
```

Показывает список доступных команд, флагов и примеров.

## dcr --version

```bash
dcr --version
```

Выводит версию и target в формате:

```text
dcr <version> (<target>)
```

## Запуск без команды

Если запустить просто `dcr` без аргументов, будет выведена справка, эквивалентная `dcr --help`.
