# dcr build

Собирает проект в профиль `debug` или `release`.

## Использование

```bash
dcr build
dcr build --debug
dcr build --release
```

## Что делает команда

- Проверяет наличие `dcr.toml`.
- Создает `target/<profile>/` при необходимости.
- Компилирует `src/main.c` через компилятор `clang`.
- Кладет бинарник в `target/<profile>/main`.

## Профили

- `--debug` (по умолчанию)
- `--release`

Флаги сборки описаны в разделе [Профили сборки](../reference/build-profiles/).

## Важные детали

- В текущей реализации разбирается только первый аргумент после `build`.
- Неизвестный аргумент или флаг приводит к ошибке.
- Если `src/main.c` отсутствует, команда не компилирует код.
